TermColors

Bring life to your terminal!

Bash setup with colorized prompt, text, dircolors, cat (bat) syntax highlighting and nano colors. Random color prompt generator by typing 'random' in your terminal. XinaA15 support! Added the ability to switch to root user with 'toor' command (root backwards).

• Color prompt

• Text colors

• Directory colors

• Syntax highlighting for cat'ing files (uses bat automatically)

• Syntax highlighting for nano editor

• Random color prompt generator (type 'random' or permanently make it random each startup)

• Now supports XinaA15 jailbreak using NewTerm3

• Switch to root using 'toor' command (root backwards)
