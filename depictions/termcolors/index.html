TermColors

Bring life to your terminal!

Bash and Zsh setup with colorized prompt, text, dircolors, cat (bat) syntax highlighting and nano colors. Random color prompt generator by typing 'random' in your terminal. Supports ALL jailbreaks, both rootful and rootless, in a single package. XinaA15 and Dopamine now fully supported. Temporary 'sudo' and 'su' fix for Xina using 'toor' command.

• Supports Bash and Zsh

• Color prompt

• Text colors

• Directory colors

• Syntax highlighting for 'cat'ting files (uses 'bat' automatically)

• Syntax highlighting for nano editor

• Random color prompt generator (type 'random' or permanently make it random each startup)

• Handy bash environment variables, like PS1 prompt directory shortening with DIRTRIM to save screen real estate

• Supports all jailbreaks, rootful and rootless, including XinaA15 and Dopamine

• Switch to root using 'toor' command (root backwards)

• Temporary sudo fix for Xina users by aliasing sudo and su to 'toor'

• Convenience fixes for a variety of common rootless issues on Xina and Dopamine such as perl lang, curl, wget, and ssl cacert pathing
