Glif

Glyph style themes. Mix and Match Icon Sets. Choose 1 or use all together. Arrange the priority order of Icon sets 1, 2, 3, 4 etc in Snowboard when used together. Highest priority goes to top of order arrangement. Best to use all together to fill the gap of missing icons. Use Glif icons by themself or with any of the included Icon BG colors with shadow AE effect. Enable only 1 BG color at a time.

Features

  • 5000+ Glyph Icons
  • Various icon sets that can be mixed and matched with each other using Snowboard priority arrangement
  • Settings Icons
  • Respring Animation
  • Boot Logo
  • Custom Clock Icon
  • Multiple Icon BG colors with Shadow AE effects
  • Plus More!
