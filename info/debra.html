Debra

Deb Repackaging Assistant. Simply type 'debra' into a terminal.

Features

  • Note: Best if ran as root
  • Easily select from list of installed Tweaks
  • Repack installed Tweaks into .deb
  • Uninstall existing Tweaks
  • Easily select from list of saved Debs
  • Extract saved Deb from CyDown folder
  • Install saved Deb from CyDown folder
  • Display installed Tweak count and saved Deb count

Known Issues

  • Layout may be distorted depending on screensize if font is too big
